Формат входных данных

Вводится числа n, m, w, r, где n -- количество городов, m -- количество предметов,  
w -- максимальная вместимость рюкзака, r -- стоимость аренды рюкзака за единицу времени.
В следующей строке указано число k -- количество дорог между городами.
Каждая из следующих k строк содержит три числа a, b, d, которые означают, что между городами a и b существует дорога длины d.

Далее вводятся m строк
В i-ой строке указаны три числа p_i, w_i и n_i -- стоимость предмета, его вес и в каком городе он находится.

Формат выходных данных

Вывести максимальное значение функции Z(П, P) (см. Benchmark2014)
